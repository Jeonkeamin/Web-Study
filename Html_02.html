    CSS stands for Cascading Style Sheets.

    CSS는 inline, Internal, External 이 세개의 종류로 되어있다.
    inline은 html 엘리먼트의 Attribute에 스타일을 넣는 것이고
    internal은 head영역에 style엘리먼트를 넣는 것이고
    External은 CSS파일을 새로 만드는 것이다.
    대부분은 External을 쓴다.
